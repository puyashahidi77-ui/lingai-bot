# LingAi Bot – Bale Voice Feedback Bot

A serverless FastAPI bot running on Vercel using:
- Bale Bot API
- Groq Whisper (STT)
- Groq Llama3 (LLM Feedback Engine)

## Deploy
1. Upload files to GitHub repo.
2. Clone with Vercel:
   https://vercel.com/new/clone?repository-url=YOUR_REPO_URL
3. Add environment variables:
   - BOT_TOKEN
   - GROQ_API_KEY
   - APP_URL
